# WordLister 1.0
Wordlister 1.0 is the smart tool to edit and manage wordlists.

Modules:-

1. Extract From ULP From File 
2. Email To User Convert 
3. Convert To Emails Wordlist Type 
4. Sort Files By Domain (Creates Seperate Files For All Domains) 
5. Remove Accounts Of Given Domain 
6. Remove Duplicate Mails Accounts 
7. Remove Accounts With Typed Passwords 
8. Remove Accounts With Wrong Mails 
9. Remove Wrong Format Accounts
10. Extract ULP From Folder
